# 🚀 My Portfolio - React + TypeScript + Vite

This is my personal portfolio website built with modern web technologies. It showcases my work, skills, and experience as a developer.

## 🛠️ Built With

- **React 18** - UI Library
- **TypeScript** - Type Safety
- **Vite** - Build Tool & Dev Server
- **React Router DOM** - Navigation
- **Tailwind CSS** - Styling (if you're using it)

- 🔍 SEO friendly
- 🚀 Optimized production build

## 📦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm/yarn/pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/y-ro9/resume-portfolio.git

# Navigate to project directory
cd resume-portfolio

# Install dependencies
npm install

# Start development server
npm run dev

Build for Production
bash

npm run build
